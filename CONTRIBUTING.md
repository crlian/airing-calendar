# Contributing

Thank you for your interest in contributing. This project is open to issues and
pull requests that improve stability, performance, and user experience.

## Guidelines
- Keep changes focused and scoped to a single improvement.
- Prefer small, reviewable pull requests.
- Follow existing code style and patterns.
- Avoid adding new dependencies unless necessary.
- Do not include secrets or private keys in commits.

## Development
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build: `npm run build`

## Submitting Changes
- Open an issue for larger changes before starting.
- Use clear commit messages.
- Include screenshots for UI changes.

## Code of Conduct
Be respectful and constructive in all interactions.
